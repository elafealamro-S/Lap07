# Lab 07 – Methods I

## 1. Objective
The purpose of this Lab is to familiarize students with:
1. Creating their own methods in Java
2. Reusing existing methods

## 2. Lab Learning Outcomes (LLO)
By completion of the lab, the students should be able to:
1. Define new methods
2. Invoke a method
3. Pass parameters by value to a method
4. Use the returned value of a method

## 3. Theory Review – Methods in Java
A method in Java is a block of code that performs a specific task and can be reused.

### Syntax for defining a method:
```java
public static <return type> <methodName>(<datatype param1, datatype param2, ...>) {
    // method body
    return value;
}
